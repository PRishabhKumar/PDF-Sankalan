Frequently Asked Questions

Is PDF Merger free to use?

Yes! PDF Merger is completely free to use with no hidden costs or limitations.

Are my files secure?

Absolutely. All processing happens locally in your browser. No files are uploaded to our servers.

What file formats are supported?

Currently, we support PDF files only. We're working on adding support for other formats.

Is there a file size limit?

Each individual PDF file can be up to 16MB in size. There's no limit on the number of files you can merge.

Can I use this on mobile devices?

Yes! PDF Merger is fully responsive and works great on desktop, tablet, and mobile devices.

Do I need to create an account?

No account required! You can start using PDF Merger immediately without any registration.
